# AI Chatbot 🤖  

A simple AI-powered chatbot using OpenAI's GPT-4 API.  

## Features  
✔️ Accepts user input  
✔️ Generates AI-powered responses  
✔️ Clean and simple UI  

## Technologies Used  
- Node.js  
- Express.js  
- OpenAI API  
- EJS for templating
